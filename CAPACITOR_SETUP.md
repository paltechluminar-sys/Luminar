# Capacitor Setup Guide - Multi-Platform Development

Your Luminar app is now set up to run on **Web, iOS, Android, and Desktop** using a single React codebase!

## 📁 Project Structure

```
luminar/
├── src/                          # Your React source code (shared across all platforms)
├── public/                       # Static assets
├── build/                        # Production build (generated by `npm run build`)
├── ios/                          # iOS native project (Xcode)
├── android/                      # Android native project (Android Studio)
├── capacitor.config.ts           # Capacitor configuration
└── package.json                  # Scripts for all platforms
```

## 🚀 Quick Start Commands

### Web (Browser)
```bash
npm start              # Development server with hot reload
npm run build          # Production build
npm run web            # Alias for build
```

### iOS (Mac required)
```bash
npm run cap:run:ios    # Build web app + sync + open in Xcode simulator
npm run cap:open:ios   # Open Xcode project
npm run cap:sync:ios   # Sync web assets to iOS only
```

### Android
```bash
npm run cap:run:android    # Build web app + sync + open in Android emulator
npm run cap:open:android   # Open Android Studio project
npm run cap:sync:android   # Sync web assets to Android only
```

### Mobile (Both)
```bash
npm run mobile         # Build and prepare for both iOS & Android
npm run cap:sync       # Sync changes to both platforms
npm run cap:build      # Build + sync for all platforms
```

## 📱 Workflow

### Development
1. **Edit React code** in `src/` like normal
2. **Run `npm start`** to test in browser with hot reload
3. **Deploy to mobile when ready:**
   ```bash
   npm run cap:run:ios      # or cap:run:android
   ```

### Production Deploy
1. **Build React app:**
   ```bash
   npm run build
   ```

2. **Sync to platforms:**
   ```bash
   npm run cap:sync         # Both iOS & Android
   # OR individual platforms:
   npm run cap:sync:ios
   npm run cap:sync:android
   ```

3. **Open native IDEs to build:**
   ```bash
   npm run cap:open:ios      # Opens Xcode
   npm run cap:open:android  # Opens Android Studio
   ```

## 🔌 Available Plugins

Your project includes these Capacitor plugins for native features:

| Plugin | Use Case |
|--------|----------|
| `@capacitor/splash-screen` | Custom splash screens |
| `@capacitor/status-bar` | Control status bar styling |
| `@capacitor/camera` | Access device camera |
| `@capacitor/geolocation` | GPS/location services |
| `@capacitor/network` | Check network connectivity |

### Using Plugins in React
```javascript
import { Camera } from '@capacitor/camera';
import { Geolocation } from '@capacitor/geolocation';

// Example: Take a photo
const takePhoto = async () => {
  const image = await Camera.getPhoto({
    quality: 90,
    allowEditing: true,
    resultType: 'uri'
  });
};

// Example: Get location
const getLocation = async () => {
  const coordinates = await Geolocation.getCurrentPosition();
  console.log(coordinates);
};
```

## ⚙️ Configuration

### App Settings (capacitor.config.ts)
Edit this file to customize app behavior:
- `appId`: Unique identifier (com.luminar.app)
- `appName`: Display name (Luminar)
- `webDir`: Build output folder (build)

### Plugins Configuration
To add plugin configurations, edit `capacitor.config.ts`:

```typescript
const config: CapacitorConfig = {
  // ... other config
  plugins: {
    Camera: {
      // Camera plugin settings
    },
    SplashScreen: {
      launchAutoHide: true,
    },
  },
};
```

## 📱 Platform-Specific Code

If you need platform-specific logic (iOS vs Android vs Web):

```javascript
import { Capacitor } from '@capacitor/core';

const platform = Capacitor.getPlatform();

if (platform === 'ios') {
  // iOS-specific code
} else if (platform === 'android') {
  // Android-specific code
} else if (platform === 'web') {
  // Web-only code
}
```

## 🛠️ Troubleshooting

### Changes not appearing on mobile?
```bash
npm run build           # Rebuild web app
npm run cap:sync       # Resync to platforms
```

### Need to add a new plugin?
```bash
npm install @capacitor/plugin-name --save
npx cap sync
```

### Clear caches and rebuild:
```bash
rm -rf build           # or del /s /q build on Windows
npm run build
npm run cap:sync
```

## 📦 Building for App Stores

### iOS App Store
1. Have a Mac with Xcode installed
2. Apple Developer account
3. Run `npm run cap:open:ios`
4. In Xcode: Set up signing, create App ID, archive, and upload

### Google Play Store
1. Android SDK installed
2. Google Play Developer account
3. Run `npm run cap:open:android`
4. In Android Studio: Create signed APK/Bundle and upload
